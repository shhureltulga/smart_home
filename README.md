# Smart Home App (Starter)

This is a clean Flutter starter skeleton using GetX + GetStorage.

## Quick start
1. Unzip this folder.
2. In the project root, run:
   ```bash
   flutter create .
   flutter pub get
   flutter run
   ```
3. You can now add your features and UI.

## Structure
```
lib/app/
  main.dart
  routes.dart
  theme.dart
  core/

  data/
    storage_service.dart
  modules/

    auth/

```
